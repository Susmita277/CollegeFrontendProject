There are several types of list in html,include:

Unordered List(ul)

It is a bulleted list where the order of the items doesn't not matter

  • Mango
  • Litchi
  • Watermelon

Ordered List(ol)

It is a numbered list where the order of the items matter

  1. Sunday
  2. Monday
  3. Tuesday

Definition List(dl)

It consists of a series of terms (dt) and their definitions (dd)

HTML:
HTML stands for HyperText Markup Language.
CSS:
CSS stands for Cascading Style Sheets.

Nested List

A nested list is a list that is contained within another list. It is a hierarchical structure where a list item can have its own sublist

  • Lalitpur
  • Kathmandu
    • Bagbazar
    • Baneshwor
  • Bhaktapur
